## Ejercicio resuelto Fuego de Quasar

## Install server test
####Prerequisites
* composer
* php >7
#### Install
* composer install 
* php artisan key:generate
## Run server
* php artisan serve
* http://127.0.0.1:8000
## Run test
* vendor/bin/phpunit tests/
